### Requirements

* Linux (Tested on Ubuntu 16.04 LTS / Windows Subsystem for Linux)
* CERN ROOT (Tested on v6.10/08)

### Download Link

Via Git:

```
git clone https://github.com/divaldo95/BrownM4.git
```

Direct link:

```
https://github.com/divaldo95/BrownM4/archive/master.zip
```
### Build

In the directory of program (BM1D), open a terminal window and enter:

```
chmod 744 setup.sh
```
After giving the necessary permissions for setup script, run:

```
./setup.sh
``` 

### Running the simulation

After build, in the directory of build (bm1d_build), open a terminal window and enter:

```
./BM1D 
```
